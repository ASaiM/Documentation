ASaiM web interface 
###################

The third part of ASaiM project is web interface available at `http://g2im.u-clermont1.fr/asaim/ <http://g2im.u-clermont1.fr/asaim/>`_. The web interface is dedicated to interactions with database: data submission, exploration, query, comparison, ...

Currently, the web interface is under construction.
