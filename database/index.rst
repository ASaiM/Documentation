.. _database:

ASaiM database 
##############

Data about intestinal microbiota are available in public data repositories such as ENA, NCBI, DDBJ, ... However, these data are dispersed, not easy to identify, to query, ... Moreover, the datasets underwent different analyses. So the available results can not be compared directly.

To take an inventory of interesting databaset in public data repositories and make them comparable, an expert database is under construction. It must fill the following 
requirements: 

 - Containing information which are standardized, easy to query and to compare (comparative analysis between different dataset)
 - Being easy to query
 - Being not too massive for long-term storage and fast query
 - Being regularly updated with new data saved in public data repositories
 - Being easily filled with framework outputs
